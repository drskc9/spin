# 🎲 Spin (Spin Wheel & Coin Toss)

A **minimal black-and-white decision generator** built as a **single `index.html` file**.
No frameworks, no internet requests, no dependencies — just pure HTML, CSS, and JavaScript.

## Features

* 🎡 **Spin Wheel** (default: Yes / No)
* ✏️ Add **custom options** to the wheel
* 🪙 **Coin Toss** (Heads / Tails)
* 🖤 Minimal **developer-style black & white UI**
* ⌨️ Font: **Hack**
* 📦 Runs completely **offline**

## Usage

1. Clone the repo

```
git clone https://github.com/drskc9/spin.git
```

2. Open the file

```
index.html
```

3. Spin the wheel or toss the coin to make a random choice.
